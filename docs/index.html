技能與掛鉤

一個負責教學，一個負責提醒

技能

`coreutils-windows-native-tools` 像小手冊。它教 Codex 何時用 `grep.exe`、`cat.exe`、`wc.exe`、`xargs.exe`。

掛鉤

`coreutils-shell-guard` 像警示燈。工作開始或 shell 用完後，提醒可能有 alias、管線或 UTF-8 風險。

安全邊界

掛鉤只提醒，不改寫命令，不改 `$PROFILE`，不改系統 PATH，也不隱藏錯誤。

判斷摘要
「讓提醒自動出現，但把真正修改交給 Codex 明確執行。」

建議工作順序

        

          

            一、先防亂碼
            
遇到中文、Unicode、JSON、Markdown、log、重導向，先套 PowerShell UTF-8 防護。

          

          

            二、再選工具
            
需要 Coreutils 時，寫完整 `.exe` 名稱，例如 `grep.exe`、`cat.exe`、`wc.exe`。

          

          

            三、看提醒
            
掛鉤提醒不是失敗，而是提示你這次 shell 命令可能需要更穩的寫法。
